### 🚀 Currently Building

**[`eco-mcp`](https://github.com/SvnFrs/eco-mcp)** — a token-thrifty **MCP server** that makes AI coding agents *cheap*. Born from the June 2026 usage-billing shock, it swaps `grep`-and-read-whole-files for **structural code-graph retrieval**, tuned semantic search, and failures-only command output — **~97% fewer tokens** on real tasks (measured & audited, not vibes). Its *Continuous Context Integration* turns each repo into a living knowledge base, so **code review becomes context review**. Lets cheap models do real work.

**[`commit-art`](https://github.com/SvnFrs/commit-art)** — Python + Go generator that paints art onto the contribution graph. It's the reason my history claims I committed `UNIX` in 1970 → see [the gallery](https://github.com/SvnFrs/contribution-graph). 🕰️
